Overview

SIMPLE_I18N wraps the ISE i18n library for translations and locale-aware formatting.

Key Features

  • Programmatic translation registration
  • .mo file loading
  • Context-aware translations
  • Full plural form support
  • Locale-aware date/time/number/currency formatting

Quick Start

Installation

<library name="simple_i18n" location="$SIMPLE_EIFFEL/simple_i18n/simple_i18n.ecf"/>

Basic Usage

local
    i18n: SIMPLE_I18N
do
    create i18n.make
    i18n.register_translation ("hello", "Bonjour")
    print (i18n.translate ("hello"))
end

API Reference

FeatureDescription
makeUse system locale
make_with_localeUse specific locale
register_translationRegister translation
translateGet translation
translate_pluralGet plural form
format_dateFormat date
format_currencyFormat currency
